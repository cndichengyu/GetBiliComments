# B站评论爬取工具

## 项目简介

这是一个用于爬取B站（哔哩哔哩）视频评论的Python工具，支持批量爬取多个视频的评论数据，并将结果保存为CSV格式。

## 功能特性

- ✅ 支持解析b23.tv短链接，自动获取真实BV号
- ✅ 多线程并行爬取，提高效率
- ✅ 自动从cookie中提取SESSDATA
- ✅ 支持自定义爬取页数和延迟
- ✅ 评论数据保存为CSV格式，包含完整信息
- ✅ 错误处理和日志输出

## 使用方法

### 1. 准备cookie

1. 打开浏览器，登录B站账号
2. 打开开发者工具（F12），进入Network标签页
3. 刷新页面，找到任意一个请求，复制完整的cookie
4. 将cookie内容保存到项目根目录的 `bilicookie.txt` 文件中

### 2. 配置目标视频

在 `bili-re.py` 文件中，修改 `target_urls` 列表，添加你要爬取评论的视频链接：

```python
target_urls = [
    "https://b23.tv/vFRMaBL",
    "https://b23.tv/KsI4qWm",
    # 添加更多视频链接...
]
```

### 3. 运行脚本

```bash
python bili-re.py
```

### 4. 查看结果

爬取完成后，评论数据会保存到 `b站评论.csv` 文件中，包含以下字段：
- 视频BV号
- 评论ID
- 用户名
- 点赞数
- 回复数
- 内容
- 发布时间

## 配置参数

在 `get_bilibili_comments` 函数中，可以调整以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| oid | 视频BV号 | 必填 |
| type_code | 内容类型，视频为1 | 1 |
| sessdata | 登录凭证 | 必填 |
| sort | 排序方式，0-按时间，1-按热度 | 1 |
| nohot | 是否不含热评，0-包含，1-不包含 | 0 |
| ps | 每页评论数 | 20 |
| max_pages | 最大爬取页数 | 100 |
| delay | 请求间隔（秒） | 1 |
| append_mode | 是否追加模式 | False |

## 注意事项

1. 请遵守B站的使用条款，不要过度请求API
2. 建议设置合理的延迟，避免被封禁
3. 本工具仅用于学习和研究目的
4. 爬取大量数据可能会触发B站的反爬机制

## 常见问题

### Q: 提示"未能从文件中提取到 SESSDATA"

A: 请确保 `bilicookie.txt` 文件中包含完整的cookie，并且其中包含 SESSDATA 字段。

### Q: 爬取失败或返回错误

A: 可能的原因：
- cookie已过期，请更新cookie
- 请求过于频繁，被B站限制
- 网络连接问题

### Q: 评论数据不完整

A: 可能是因为：
- 视频评论过多，达到了 `max_pages` 限制
- 部分评论被B站隐藏或删除
